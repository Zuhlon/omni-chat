# 🎧 Омниканальный Мессенджер - Аудиозал Hi-End

Интерактивный прототип мессенджера для записи в аудиозал и покупки hi-end аудиотехники.

## 📱 Экраны прототипа

### Экран 1: Запись на прослушивание
- Переписка о записи в аудиозал
- Голосовое сообщение с транскрибацией (полный текст / резюме)
- История входящих и исходящих звонков
- Карточка записи на сеанс прослушивания

### Экран 2: Договорённости о покупке
- Переписка о покупке после прослушивания
- Коммерческое предложение с детализацией заказа
- Подтверждение заказа с доставкой

## 🎨 Особенности дизайна

- **iPhone 16 Mockup** - реалистичный макет устройства
- **Голосовые сообщения** - с транскрибацией и переключением режимов просмотра
- **История звонков** - входящие, исходящие, пропущенные
- **Интерактивные элементы** - кнопки, анимации, уведомления
- **Адаптивный дизайн** - работает на всех устройствах

## 🛠 Технологии

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- Vanilla JavaScript
- Без внешних зависимостей

## 🚀 Запуск

1. Откройте `index.html` в браузере
2. Используйте навигацию внизу для переключения между экранами

Или запустите локальный сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (если установлен http-server)
npx http-server
```

## 📞 Интерактивные функции

- **Кнопка звонка** - симуляция вызова
- **Голосовое сообщение** - кнопка воспроизведения
- **Переключатели транскрибации** - "Полный текст" / "Резюме"
- **История звонков** - раскрывающаяся панель
- **Подтверждение записи** - интерактивная кнопка
- **Отправка сообщений** - введите текст и нажмите Enter

## 📝 Тема

Прототип демонстрирует взаимодействие клиента с премиальным аудиозалом для прослушивания и покупки hi-end аудиотехники:

- **Бренды:** Wilson Audio, McIntosh, dCS, Audio Research
- **Услуги:** Прослушивание, доставка, профессиональная настройка
